Semantic Markup

<html> is the root element all content is contained between these elements.

    <table>                           Container element for for 2d table   

        <thead>                       Defines tables heading for columns

            <tr>                      Defines a table row

                <th></th>             Defines heading data for one cell,add more multiple cells

            </tr>                     closing tag for table row

        </thead>                      Closing tag for table headers

        <tbody>                       table Element container for rest of data

            <tr>                      Table Row

            </tr>

        </tbody>

        <tfoot>                       Table footer
